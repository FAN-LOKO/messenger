# Roles and Permissions

> RU: Описание ролей и базовой логики прав доступа.
> EN: Description of roles and core access permission logic.

## Roles
- user
- moderator
- admin
- premium

## Base logic
Premium is a product status layer.
Moderator and admin are system roles.

## Permissions

### user
- read feed
- participate in chats
- react and comment
- access locked content only after unlock
- join calls only if invited and allowed by policy

### premium
- all user permissions
- publish posts
- create 1:1 video calls
- join premium group calls
- access premium content
- use token-based features

### moderator
- moderate chats
- moderate feed posts
- hide content
- pin content
- control community quality

### admin
- full moderation access
- create 1:1 and group calls
- manage rooms
- manage premium logic
- manage featured content
- manage policies and permissions
